# AI Fact Verification & Intelligence Platform

AI-powered misinformation detection and claim verification platform built using Streamlit, OpenRouter, and Tavily APIs.

## Features

- AI claim verification
- Real-time web evidence retrieval
- Confidence scoring
- Source credibility analysis
- Fake news risk detection
- Analytics dashboard
- Interactive visualizations

## Tech Stack

- Python
- Streamlit
- OpenRouter API
- Tavily Search API
- SQLite
- Plotly
